============================================================
Module su/ (SU(N) group and Haar interface)
============================================================
Hypotheses
- N ∈ ℕ, N ≥ 2. SU(N) is the special unitary group: SU(N) = {U ∈ M_N(ℂ): U* U = I, det U = 1}.
- Compact Lie group; normalized Haar probability measure μ_Haar on SU(N).

Statements
- Existence and uniqueness of normalized Haar measure μ_Haar.
- Basic integration invariances: ∫ f(g) dμ = ∫ f(hg) dμ = ∫ f(gh) dμ.

Constants/bounds
- For any matrix entry |(U)_{ij}| ≤ 1; |Tr(U)| ≤ N.
- For plaquette factors: 0 ≤ 1 − (1/N) Re Tr(U) ≤ 2.

Inequality sources
- Compactness ⇒ Haar exists, is unique (standard).
- Cauchy–Schwarz ⇒ |Tr(U)| ≤ √N · ∥U∥_HS = N.

Lean lemma names (targets)
- su.haar_normalized : ExistsUnique (ProbabilityMeasure SU(N)).
- su.integral_left_invariant, su.integral_right_invariant.

Module ym_measure/ (Wilson action and Gibbs measure)
============================================================
Hypotheses
- Configuration U = {U_{x,μ}} ∈ ∏_{(x,μ)∈E} SU(N).
- Wilson action S_β(U) = β Σ_{P} (1 − (1/N) Re Tr(U_P)).
- Product Haar measure dU = ⊗ dμ_Haar(U_{x,μ}).

Statements

Constants/bounds
- 0 ≤ S_β(U) ≤ 2β |P|, hence e^{−2β|P|} ≤ e^{−S_β(U)} ≤ 1.
- Z_β ∈ [e^{−2β|P|}, 1].

Inequality sources
- SU(N) compactness and 0 ≤ 1 − (1/N) Re Tr(U_P) ≤ 2.

Lean lemma names (targets)

Module os_pos_wilson/ (OS positivity for Wilson via link reflection)
============================================================
Hypotheses
- Choose a time-reflection θ across a mid-plane between t=0 and t=1 time-slices (link-reflection).
- Observables F supported in positive time half (t ≥ 1).
- Measure dμ_β as above; action local and reflection-symmetric.

Statements
- Reflection positivity: for finite set {F_i} of cylinder observables in t ≥ 1,
  the Gram matrix G_{ij} = ⟨(θF_i)^* F_j⟩_β is PSD.
- Existence of positive self-adjoint transfer T realizing one Euclidean time step.

Constants/bounds
- Holds for all β > 0 and all N ≥ 2 (no small-coupling restriction needed for RP).

Inequality/identity sources
- Osterwalder–Seiler link-reflection factorization: action splits across the cut; the inter-slab integral defines a positive kernel.
- Positivity from group-integration: ∫ φ(g)^* K(g,h) φ(h) dμ ≥ 0 when K is of the form ∫ A(g,u)^* A(h,u) dν(u).

Lean lemma names (targets)
- os_wilson_reflection_psd : OSPsd θ μ_β.
- os_wilson_transfer_positive_selfAdjoint : Transfer positivity/self-adjointness.

Module gap_strong_coupling/ (finite-volume spectral gap at strong coupling)
============================================================
Hypotheses
- Strong coupling regime: β is sufficiently small.
- Inter-slab interactions across the OS cut have total coupling J_⊥ (geometric constant from lattice).
- Observables normalized to zero mean in OS inner product (orthogonal to Ω).

Statements
- Dobrushin influence bound: c_{xy} ≤ tanh(β J_{xy}) ≤ 2β J_{xy}.
- Cross-cut contraction: α_⊥ := sup_{x∈L} Σ_{y∈R} c_{xy} ≤ 2β J_⊥.
- Correlation decay: |(F, T^t F)_OS| ≤ ∥F∥^2 α_⊥^t.
- Spectral radius on mean-zero subspace: r0(T) ≤ α_⊥ ≤ 2β J_⊥.
- Gap: γ(β) := −log r0(T) ≥ −log(2β J_⊥).

Constants/bounds
- Take explicit smallness: β ≤ 1/(4 J_*), with J_* := sup_x Σ_y J_{xy} ⇒ 2β J_⊥ ≤ 1/2 ⇒ γ(β) ≥ log 2.
- N-uniformity: bounds use only unit-norm assumptions and ferromagnetic sign, not N.

Inequality sources
- Radon–Nikodym derivative bound for single-site conditionals: e^{−2β J_{xj}} ≤ dμ_x/dμ'_x ≤ e^{+2β J_{xj}}.
- Total variation ≤ tanh(β J_{xj}) ≤ 2β J_{xj}.
- Spectral theorem: r0(T)^t = sup_{∥F∥=1} |(F, T^t F)|.

Lean lemma names (targets)
- dobrushin_influence_bound, crosscut_alpha_bound, transfer_radius_leq_alpha, gap_lower_bound_from_alpha.

Module rs_vs_wilson/ (RS↔Wilson kernel comparability in a scale window)
============================================================
Hypotheses
- Kernels K_RS, K_W on loop index set Γ with shared symmetries (time-translation, reflection).
- Locality: |K_X(γ,γ')| ≤ A_X e^{−μ_X d(γ,γ')} (X ∈ {RS, W}).
- On-site: b_X ≤ K_X(γ,γ) ≤ B_X; Growth: # {γ': d(γ,γ') = r} ≤ C_g e^{ν r}.
- μ_X > ν; define S(α) = C_g/(e^{α − ν} − 1); S_X := S(μ_X).

Statements
- Gershgorin bounds: (b_X − A_X S_X) ∥v∥^2 ≤ v*Gram_X v ≤ (B_X + A_X S_X) ∥v∥^2.
- Comparability: c1·Gram_W ≤ Gram_RS ≤ c2·Gram_W with
  c1 = (b_RS − A_RS S_RS)/(B_W + A_W S_W), c2 = (B_RS + A_RS S_RS)/(b_W − A_W S_W).
- OS version holds with θ-reflection Gram.

Constants/bounds
- β0^{OS}(K_X) ≥ b_X − A_X S_X > 0.

Inequality sources
- Schur/Gershgorin diagonal dominance using locality and growth bounds.

Lean lemma names (targets)
- kernel_gram_bounds, rs_wilson_gram_comparability, os_comparability.

============================================================
Hypotheses
- Operators T_n → T in operator norm (or via norm–resolvent for H_n → H).
- Each T_n has gap ≥ γ > 0 on Ω^⊥.

Statements

- Gap persistence: if ∥T_n − T∥ ≤ ε < 1 − e^{−γ}, then spec(T|_{Ω^⊥}) ⊆ [0, e^{−(γ−δ)}] with δ = −log(1−ε).
- Norm–resolvent ⇒ spectral inclusion and β0 preservation (R3).

Constants/bounds
- δ controlled by norm difference; choose ε from construction.

Inequality sources
- Min–max principle and Weyl inequalities; second resolvent identity.

Module continuum_limit/ (R3 implementation and continuum gap)
============================================================
Hypotheses
- Embeddings I_a : H_a → H; projections P_a = I_a I_a^*.
- Defects ε_a := ∥(H I_a − I_a H_a)(H_a + 1)^{−1/2}∥ → 0; compact (H − z0)^{−1}.
- inf spec(H_a) ≥ β0 > 0 uniformly.

Statements
- Quantitative resolvent bound (for nonreal z0):
  ∥(H − z0)^{−1} − I_a(H_a − z0)^{−1}I_a^*∥ ≤ δ_a(Λ)/dist(z0,[0,Λ]) + η(Λ; z0) + C(β0,z0) ε_a,
  where δ_a(Λ)=∥(I−P_a)E_H([0,Λ])∥, η(Λ; z0)=sup_{λ≥Λ} 1/|λ−z0|.
- For all nonreal z, convergence via resolvent identity; spec(H) ⊆ [β0,∞).

Constants/bounds
- Choose Λ s.t. η(Λ; z0) small; then a large so that δ_a(Λ) and ε_a small.
- C(β0,z0) := ∥(H − z0)^{−1}∥ sup_{λ≥β0} √(1+λ)/|λ−z0|.

Inequality sources
- Master identity: R(z) − I R_a(z) I^* = R(z)(I − P_a) − R(z) D_a R_a(z) I^*.
- Neumann series and functional calculus.

Lean lemma names (targets)
- continuum_resolvent_bound, continuum_norm_resolvent, continuum_spec_lower_bound.

Renormalization_and_units/ (β(a) scaling, units, Δ_phys)
============================================================
Hypotheses
- Spatial lattice spacing a_s>0; temporal spacing τ>0 (allow anisotropy; isotropic case a_s=τ=a).
- Transfer T := e^{−τ H_phys}. Lattice spectral radius bound r0(β) on Ω^⊥ and lattice gap γ_latt(β):=−log r0(β).
- Cross‑cut constant J_⊥ independent of L and a (geometric/local).

Statements
- Physical mass gap Δ_phys(β,τ) := γ_latt(β)/τ.
- Choose scaling schedule β=β(τ) so that α_⊥(β(τ)) = e^{−τ Δ0} with Δ0>0 fixed, and r0(β) ≤ α_⊥(β).
  Using α_⊥(β) ≤ 2 β J_⊥, suffices to take β(τ) = (1/(2 J_⊥)) e^{−τ Δ0}.
  Then r0(β(τ)) ≤ e^{−τ Δ0} and Δ_phys(β(τ),τ) ≥ Δ0 uniformly as τ→0.

Constants/bounds
- For small τ: β(τ) = (1/(2 J_⊥)) (1 − τ Δ0 + O(τ^2)) < 1/(2 J_⊥).
- Uniform in N: J_⊥, the Dobrushin bound, and the transfer calculus do not depend on N.

Inequality sources
- r0(T) ≤ α_⊥(β) ≤ 2 β J_⊥ and T=e^{−τ H_phys} ⇒ r0=e^{−τ Δ_phys}.

Lean lemma names (targets)
- scaling_beta_schedule, physical_gap_from_schedule : Δ_phys ≥ Δ0.
